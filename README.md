# ShopHub E-commerce Website

A modern, responsive e-commerce website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Mobile Hamburger Menu** - Toggle between ☰ and ✕ icons
- **Shopping Cart** - Add, remove, and update product quantities
- **Image Hover Effects** - Smooth zoom and "View" overlay on hover
- **Smooth Scrolling** - Navigation links scroll smoothly to sections
- **Kenyan Shilling (KSh) Pricing** - Localized currency display

## Project Structure

```
E-commerce/
├── index.html    # Main HTML file
├── styles.css    # CSS styles
├── script.js     # JavaScript functionality
└── README.md    # This file
```

## Sections

1. **Header** - Navigation with logo and menu
2. **Hero** - Welcome banner with CTA button
3. **Categories** - Product categories (Fashion, Electronics, Home & Garden, Sports)
4. **Featured Products** - 4 products with images, prices, and Add to Cart buttons
5. **Our Projects** - 3 project showcase cards
6. **Shopping Cart Modal** - Cart management interface

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/kevinwangombe/e-commerce-2.git
   ```

2. Open `index.html` in your browser
